## About Me

Started in 2021 building custom ROMs for Android devices that manufacturers had abandoned, learning hands-on, contributing to open-source communities, and getting hooked on seeing real users run what I built. Spending hours on something that could still break and not minding was what told me this was the right path. Today I apply that same hands-on approach to production systems in health tech and academic research, still most motivated when the work ends up in real people's hands.
